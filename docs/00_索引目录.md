## 🎓 学习建议

### 对于完全新手：
1. **第1天**：阅读 01、02 文档，建立基础认知
2. **第2天**：阅读 03 文档，理解算法原理
3. **第3天**：跟着 04 文档完成第一次训练
4. **第4天**：阅读 05 文档，深入代码细节
5. **第5天**：完成 06 文档的所有练习

### 对于有编程基础的学习者：
1. **快速浏览** 01、02 文档（30分钟）
2. **重点阅读** 03 文档，理解算法（1小时）
3. **实战训练** 04 + 06 文档（2-3小时）
4. **深入代码** 05 文档（可选，1小时）

### 对于只想快速使用的用户：
直接跳转到 [04_训练全流程.md](./04_训练全流程.md) 的"快速开始"部分
